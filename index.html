¿Cuál es el problema de las N reinas?

El problema trata de ubicar un número N de reinas en un tablero de N x N de manera tal que no se ataquen entre sí, es decir, dos reinas no pueden estar en la misma fila, columna o diagonal.

En esta página puedes ver como el algoritmo de backtracking intenta encontrar una solución válida, realizando pruebas y retrocediendo cuando se encuentra una solución inválida.
